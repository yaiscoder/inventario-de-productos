# inventario de productos

## link del repositorio: https://github.com/yaiscoder/inventario-de-productos.git

## Descripción
Este programa permite registrar y consultar un inventario de productos, en donde se podrá agregar un nuevo producto, consultar los que estén disponibkes en el inventario y calcular una estadistica de este, tanto el precio total como la cantidad total de productos que están en el inventario.

## Funcionamiento
- Es un programa de consola.
- Funciona en "Python" a través de un intérprete (por ejemplo: Python 3).
- Permite registrar productos con las siguientes variables:
  - "nombre: nombre del producto.
  - precio: precio del producto.
  - cantidad: número de productos disponibles en el inventario.
- Permite imprimir el inventario
- Permite calcular la estadistica del inventario, mostrando:
  - Precio Total de todo el inventario
  - Cantidad de productos existentes en el inventario

## Requisitos
- Tener "Python 3" instalado
- Un intérprete o terminal para ejecutar el programa

1. Descargar o clonar el repositorio
