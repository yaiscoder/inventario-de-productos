# inventario de productos

## link del repositorio: https://github.com/yaiscoder/inventario-de-productos.git

## Descripción
Este programa permite registrar y consultar un inventario de productos mediante un menú interactivo por consola, realizando operaciones como CRUD (crear, leer, actualizar, eliminar), calculo de estadística y persistencia de datos mediante archivos csv.

## Funcionamiento
- Es un programa de consola.
- Funciona en "Python" a través de un intérprete (por ejemplo: Python 3).
- Permite registrar productos con las siguientes variables:
  - "nombre: nombre del producto.
  - precio: precio del producto.
  - cantidad: número de productos disponibles en el inventario.
- Permite imprimir el inventario.
- Buscar productos en el inventario.
- Actualizar información de un producto.
- Eliminar un producto.
- Permite calcular la estadistica del inventario, mostrando:
  - Precio Total de todo el inventario.
  - Cantidad de productos existentes en el inventario.
- Guardar inventario en CSV.
- Cargar inventario desde CSV.
- Salir del menú.

## Requisitos
- Tener "Python 3" instalado.
- Un entorno de desarrollo (IDE) como Visual Studio Code.

## Cómo ejecutar el programa: 
1. Descargar o clonar el repositorio.
2. Cambiar la rama de entrada a "main".
3. Ejecutar el archivo principal (app.py).
4. Seguir las instrucciones del menú.
